# pinball-physics
Some 2D rigid body dynamics made for Ubisoft Toronto's [NXT showcase](http://toronto.ubisoft.com/ubisoft-toronto-nxt-showcase/).

Ball-plane, ball-ball, and ball-flipper collisions are handled.
